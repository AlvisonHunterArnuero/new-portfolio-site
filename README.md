# Welcome to my new site - Enhanced version 2026

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Note
To update the site that is fed with this content, we run pnpm run build, get the dist folder files after completing the build and then paste it in the main repository folder of this github account to update the personal site.
